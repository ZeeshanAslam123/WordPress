Easily connect LearnDash with GamiPress! Add GamiPress-specific triggers to LearnDash courses, lessons, topics, and quizzes right from the backend editor.

== Description ==

**Swrice’s GamiPress LearnDash Trigger Integration** is a powerful plugin designed to seamlessly integrate GamiPress triggers into your LearnDash-powered LMS.

With this add-on, you can reward learners with points based on their interactions with courses, lessons, topics, and quizzes — all managed directly from the WordPress backend. It’s a simple yet highly effective way to gamify your e-learning experience and boost engagement.

### Key Features

- Add GamiPress triggers from the edit screen of LearnDash **Courses**, **Lessons**, **Topics**, and **Quizzes**
- Select **point type** (e.g., experience, achievement)
- Set the **number of points** to award
- Define a **limited time option**
- Set a **maximum earning limit**
- Add a **custom label** for the trigger

### Plugin Requirements

- WordPress 5.5 or higher
- PHP 7.0 or higher
- Compatible with all GamiPress versions
- Works with any LearnDash version

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory or install it via the WordPress Plugin Installer.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to any Course, Lesson, Topic, or Quiz in LearnDash.
4. Use the new "GamiPress Trigger" metabox to configure your desired settings.

== Frequently Asked Questions ==

= Does this plugin work with any version of GamiPress? =  
Yes, it is fully compatible with all GamiPress versions.

= Will this plugin work with my current LearnDash setup? =  
Absolutely. It supports all versions of LearnDash and integrates directly into the course, lesson, topic, and quiz edit screens.

= Can I set a limit to how many points a user can earn? =  
Yes, there is an option to define the **maximum earning points** in the trigger settings.
