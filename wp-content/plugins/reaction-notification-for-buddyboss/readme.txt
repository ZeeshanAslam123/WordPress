== Description ==

Swrice presents **Reaction Notification for BuddyBoss**, a lightweight plugin that enhances user engagement on your BuddyBoss-powered community.

This plugin automatically sends a notification to the post author whenever another user likes or reacts to their activity post. You can also customize how reactions are displayed in the notifications: show the reaction name, icon, or both.

== Features ==

✅ Sends notifications when a user likes or reacts to another user's activity post.  
✅ Lets you choose to display the reaction name, icon, or both in notifications.  
✅ Lightweight, clean, and seamlessly integrates with BuddyBoss.  
✅ Easy to set up and manage from your BuddyBoss admin panel.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to **BuddyBoss > Settings > Reaction Notifications** to configure your display preferences.
4. Done! The post author will now receive notifications when someone reacts to their activity post.
