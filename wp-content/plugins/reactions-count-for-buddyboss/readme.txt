== Description ==

**Reactions Count for BuddyBoss** by Swrice offers a seamless way to show live reaction counts BuddyBoss activity posts. Designed for simplicity and performance, this plugin enables real-time engagement feedback directly within the activity feed.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/reactions-count-for-buddyboss` directory, or install the plugin via the WordPress plugin repository.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Navigate to **BuddyBoss > Reaction Count Settings** to enable or disable reaction count display.

== Frequently Asked Questions ==

= Does this plugin support all BuddyBoss themes? =  
Yes, it is designed specifically for full compatibility with the BuddyBoss platform and its official themes.

= Are reaction counts updated live without page refresh? =  
Yes! Reaction counts update dynamically via AJAX when users react to posts.

= Can I choose which reactions to display? =  
Currently, the plugin displays total reactions including likes and comments. More granular control may be added in a future update.
