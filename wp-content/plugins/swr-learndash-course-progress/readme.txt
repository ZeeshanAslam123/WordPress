== Description ==

**Swrice LearnDash Course Progress Add-on** is a powerful and easy-to-use plugin that allows you to display student progress for any LearnDash course using either a shortcode or a Gutenberg block.

- Widget-compatible: embed in sidebars or footer widget areas

ðŸš€ **How to Use**
1. Install and activate the plugin.
2. Go to **Settings > Learndash > Progress Settings**.
3. Choose your preferred general color.
4. Use either the shortcode or the Gutenberg block to display progress:
   - **Shortcode**: `[swr_course_progress course_id=123]`
   - **Gutenberg Block**: Search for **SWR Course Progress**
5. Replace the course ID with the actual ID of your LearnDash course.
6. Paste the shortcode or insert the block into any page, post, or widget area.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/swr-learndash-course-progress/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to **Settings > Learndash > Progress settings** to configure your general color and get the shortcode or use the Gutenberg block.
4. Insert the shortcode or block into any page, post, or widget to show course progress.

== Frequently Asked Questions ==

= Can I show progress for multiple courses? =  
Yes, simply use the shortcode multiple times with different course IDs or add multiple blocks.

= Does the plugin work with all LearnDash themes? =  
Yes, the plugin is designed to be compatible with all major LearnDash themes and templates.

= Is this plugin mobile responsive? =  
Absolutely. The progress bar layout adjusts to fit all screen sizes.
